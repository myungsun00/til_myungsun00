## 변수
### 변수선언
- 변수 선언과 초기화
    ```
    int i = 0;
    ```

- 변수 선언 후 초기화
    ```
    int i;
    i = 0;
    ```

### 변수명
- 자바 키워드 사용불가
- 숫자로 시작 불가
- _와 $ 외 특수문자 사용불가

## 자료형
- 기본 자료형

        - int (4byte)
        - long (8byte)
    - 실수형
        - float(4byte)
        - double(8byte)
